# EmployeeManagement
Basic CRUD project to save/handle employee name and generate a random ID.


Made using VSCode version 1.106.1 for mac on apple sillicon.

<H1>EMPLOYEE MANAGEMENT SYSTEM</H1>


Overview:
 This project is a simple employee management system using the Pandas library for the csv/data retention and updating part on a locally stored csv file.

 Features:
  Has an admin panel to edit/remove employee ids.</br>
  Saves the data to a local CSV file in realtime so that it can be shared if needed.</br>
  Easy to understand UI. </br>
  Makes sure no single ID is assigned to two employees. </br>

Tools used:
  Pandas library to read and write to employees.csv. </br>
  random library so that each employee gets a random ID. </br>
  OS library to access file from local disk and clear screen when needed. </br>

Steps to install & run:
  Go to the top middle under code and save the repo as a zip, then extract the zip file and run untitled12.py </br>
  or clone this repository and run untitled 12.py in your python interpreter. </br>
  Made in python version 3.14.0 </br>
